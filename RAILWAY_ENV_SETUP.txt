STRIPE PAYMENT FLOW:
================================================================================

When a user pays with Stripe:

1. User clicks "Pay with Stripe" button
2. Backend creates Stripe checkout session with:
   - success_url: {FRONTEND_URL}/my-bookings?payment=success&booking_id=X&session_id=Y
   - cancel_url: {FRONTEND_URL}/my-bookings?payment=cancelled&booking_id=X

3. User completes payment on Stripe
4. Stripe redirects back to success_url or cancel_url
5. Frontend detects payment parameters and verifies with backend
6. Backend confirms payment with Stripe and updates booking status
